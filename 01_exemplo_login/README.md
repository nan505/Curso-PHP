## Criamos três página com PHP e HTML, sendo elas:

### 1. Formulário - Usuário deverá informar um e-mail e senha que serão enviados para validação via "POST"

### 2. Validar login - Valida se o usuário informou o e-mail e senha de administrador. Caso as informações sejam inválidas, retorna o usuário para a tela de formulário com uma mensagem de erro enviada via "GET".

### 3. Tela administrativa - Caso o usuário informe o e-mail e senha corretos, a página administrativa será aberta!
